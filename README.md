# 🤖 AI Customer Support Chatbot

An AI-powered customer support chatbot built using RAG (Retrieval-Augmented Generation).  
The chatbot can answer questions from uploaded PDF documents using semantic search and Large Language Models.

---

# 🚀 Features

- 📄 Multiple PDF Upload Support
- 🧠 Semantic Search using FAISS
- 🤖 Groq LLM Integration
- 💬 ChatGPT-style Streamlit UI
- ⚡ Fast Vector Search
- ✍️ Typing Animation
- 📚 Context-Aware AI Responses
- 🔥 RAG Architecture
- 🗂 Persistent Vector Database
- 🧩 Multi-document Question Answering

---

# 🛠 Tech Stack

| Category | Technology |
|---|---|
| Programming Language | Python |
| Frontend UI | Streamlit |
| LLM | Groq |
| Framework | LangChain |
| Vector Database | FAISS |
| Embeddings | HuggingFace |
| PDF Loader | PyPDFLoader |
| Environment Variables | python-dotenv |
